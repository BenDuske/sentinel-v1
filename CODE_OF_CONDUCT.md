# Code of Conduct

## Our pledge
We want participation in this project to be a harassment-free experience for everyone,
regardless of age, body, disability, ethnicity, sex characteristics, gender identity and
expression, level of experience, education, socio-economic status, nationality, personal
appearance, race, religion, or sexual identity and orientation.

## Our standards
**Encouraged:** being respectful, welcoming differing viewpoints, giving and accepting
constructive feedback gracefully, and focusing on what is best for the community.

**Unacceptable:** sexualized language or imagery; trolling, insults, or personal/political
attacks; public or private harassment; publishing others' private information without
permission; and any conduct that is illegal or that violates this project's
[Acceptable Use Policy](docs/legal/ACCEPTABLE_USE_POLICY.md).

## Scope
This Code applies in all project spaces (issues, pull requests, discussions) and when an
individual is representing the project in public spaces.

## Enforcement
Report unacceptable behavior through the repository maintainer via a GitHub issue (for
non-sensitive matters) or a [private security advisory](https://github.com/BenDuske/sentinel-v1/security/advisories/new)
(for sensitive ones). Maintainers may remove, edit, or reject contributions and comments that
violate this Code, and may ban contributors for behavior they deem inappropriate.

This Code of Conduct is adapted from the [Contributor Covenant](https://www.contributor-covenant.org),
version 2.x.
